# Homework 2 表單元素說明

本文件說明 `index.html` 中各表單欄位與 HTML 標籤的對應關係。

## 表單欄位對照表

| 介面欄位 | HTML 標籤  | Type 屬性 | 功能說明                           |
| :------- | :--------- | :-------- | :--------------------------------- |
| **姓名** | `<input>`  | text      | 基本的單行文字輸入框               |
| **帳號** | `<input>`  | text      | 單行文字輸入框，限制純英數字輸入   |
| **密碼** | `<input>`  | password  | 將輸入內容隱藏的密碼專用框         |
| **國家** | `<select>` | 無        | 提供多個選項的下拉式選單           |
| **興趣** | `<input>`  | checkbox  | 支援多重選擇的勾選框（複選）       |
| **性別** | `<input>`  | radio     | 同群組內只能擇一的單選按鈕         |
| **出生日期**| `<input>`| date      | 支援月曆介面的日期選擇器           |
| **年齡** | `<input>`  | number    | 限制只能輸入數字的數字輸入框       |
| **留言** | `<textarea>`| 無        | 支援多行輸入的文字區域             |
| **提交表單**| `<button>` | submit    | 點擊後可送出表單資料的按鈕         |
